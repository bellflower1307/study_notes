    Javaの基礎

    この章では、Javaの基本的な概念と特徴について学びます。

    Javaの特徴

    Write Once, Run Anywhere (WORA)

    JavaはプログラムとOSの間に依存関係がないため、一度書いたコードを様々な環境で実行できます。

    JVM (Java Virtual Machine)

    JavaはJVM(Java 仮想マシン)を介して実行されます。

    オブジェクト指向

    Javaはオブジェクト指向プログラミング言語であり、クラスとオブジェクトを使用してコードを構築します。

    Java環境

    JDK (Java Development Kit)

    Javaを利用してプログラムを作成したり、作成したプログラムを実行するにはJava開発環境が必要です。

    Java開発環境はJDK(Java 開発キット)で構築します。

    JDKはOracle社だけではなく、様々なベンダーから提供されており、有償のものや無償のものがあります。

    Javaのバージョン

    Javaは複数のバージョンが存在し、それぞれに異なる機能や改善が含まれています。

    年6回の定期リリースとして、毎年3月と9月に新しいバージョンがリリースされます。これをフィーチャー・リリースと呼びます。

    また、各フィーチャー・リリースには、新しい言語機能やパフォーマンスの向上、セキュリティの強化などを施したアップデート・リリースが2回含まれます。

    JRE (Java Runtime Environment)

    JRE(Java 実行環境)はJavaアプリケーションを実行するために必要な環境を提供します。

    Java 11から単体配布されなくなり、JDKに含まれるようになりました。

    また、Java環境を持たないユーザのために、配布するアプリケーションにJREが同梱されるようになりました。
