    繰り返し処理

    この章では、Javaでの繰り返し処理について学びます。繰り返し処理とは、同じ処理を複数回実行するための構文です。今までは一度だけ実行される処理でしたが、繰り返し処理を使うことで、特定の条件が満たされる間、処理を繰り返すことができます。

    Javaには、以下のような繰り返し処理の構文があります。

    • for文
    • while文
    • do-while文

    以下では、各繰り返し処理の構文と使用方法について詳しく説明します。

    for文

    for文は、指定された回数だけ処理を繰り返す構文です。

    for (初期化; 条件; 更新) {
    // 処理
}

    for文ではまず初期化が実行され、その後条件が評価されます。条件が真の間、処理が実行され、処理の後に更新が実行されます。

    初期化、条件、更新は省略可能ですが、それぞれのセミコロンは必須です。セミコロンがない場合はコンパイルエラーになります。

    初期化では宣言式（型名と変数名を記載し、初期化してある式）は1つのみ使用できます。宣言ではない式は複数記載することができます。

    条件を省略した場合は、条件が常に真と評価されます。また、最初から条件を満たさない場合は処理は1度も実行されません。

    更新も初期化と同様に複数記載することができます。

    while文

    while文は、条件が真の間、処理を繰り返す構文です。

    while (条件) {
    // 処理
}

    while文もfor文と同様に、最初から条件を満たさない場合は処理は1度も実行されません。

    do-while文

    do-while文は、少なくとも1回は処理を実行し、その後条件が真の間処理を繰り返す構文です。

    do {
    // 処理
} while (条件);

    do-while文では、まずdoブロックの処理が必ず実行されます。その後、条件が真の間処理が繰り返されます。

    拡張for文

    拡張for文は、配列やコレクションの要素を順番に処理するための構文です（配列、コレクションは後述）。

    詳細は第6章で説明します。

    制御文のネスト

    制御文のネストとは、一つの制御文の中に別の制御文を含める構造です。例えば、for文の中にwhile文を含めることもできます。

    for (初期化; 条件; 更新) {
    while (条件) {
        // 処理
    }
}

    上記の例では、for文の中にwhile文がネストされています。for文の各反復ごとに、while文が実行されます。制御文のネストは、複雑な処理を実装する際に役立ちますが、過度なネストはコードの可読性を低下させる可能性があるため注意が必要です。

    繰り返し文の他に、条件分岐もネストさせることができます。例えば、for文の中にif文を含めることもできます。

    for (初期化; 条件; 更新) {
    if (条件) {
        // 処理
    }
}

    break文

    break文は、繰り返し処理を途中で終了するための構文です。

    break;

    break文が実行されると、現在の繰り返し処理を終了し、次の処理に移ります。for文、while文、do-while文のいずれの繰り返し処理内でも使用できます。

    また、break文はswitch文内でも使用できます。switch文内でbreak文が実行されると、現在のcaseブロックを終了し、switch文全体を終了します。

    さらに、break文はラベルを指定して使用することもできます。ラベルを使用しない場合は直前の繰り返し文またはswitch文を抜けますが、ラベルを指定したbreak文が実行されると、指定されたラベルの位置まで繰り返し処理を終了します。

    label: for (初期化; 条件; 更新) {
    for (初期化; 条件; 更新) {
        // 処理
    
        if (条件) {
            break label;
        }
    }
}

    上記の例では、labelというラベルがfor文に付けられています。labelを指定したbreak文が実行されると、labelの位置まで繰り返し処理を終了します。

    continue文

    continue文は、繰り返し処理の現在の反復をスキップし、次の反復に移るための構文です。

    continue;

    continue文が実行されると、現在の反復の残りの処理がスキップされ、次の反復に移ります。for文、while文、do-while文のいずれの繰り返し処理内でも使用できます。

    また、continue文もラベルを指定して使用することができます。ラベルを使用しない場合は直前の繰り返し文をスキップしますが、ラベルを指定したcontinue文が実行されると、指定されたラベルの位置まで繰り返し処理をスキップします。

    label: for (初期化; 条件; 更新) {
    for (初期化; 条件; 更新) {
        // 処理
    
        if (条件) {
            continue label;
        }
    }
}

    上記の例では、labelというラベルがfor文に付けられています。labelを指定したcontinue文が実行されると、labelの位置まで繰り返し処理をスキップします。

    break文と違い、continue文はswitch文とは関係ありません。

    break文もcontinue文もif文と組み合わせて使用しますが、if文ではなく、if文の外の繰り返し文やswitch文を対象にしていることに注意してください。
